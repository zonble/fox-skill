# 台灣南島語系 SKILL

在這個 SKILL 當中，包含一組可以查詢台灣南島語系詞彙的工具。這個工具可以讓用戶查詢在某個台灣南島語言當中，一段中文字當中可能對應的詞彙，

AI 可以透過這個工具，查到對應的單字後，利用這些單字，讓一句原本中文的句子，寫成對應的台灣南島語系的句子。

AI 如果自己有台灣南島語系語言的知識，可以優先使用自己的能力，但是在同一個大語系下，可能有些小語系的單字不一樣，可以透過這個工具，查詢到更精確的單字。

不要使用 source 下的 json 檔案，因為這些檔案是用來建立資料庫的，並不是用來查詢的。

## 工具的限制

- 這個工具使用 Swift 寫成，並且使用 SQLite 作為資料庫。
- 這個工具用到了 macOS 的 Foundation 與 NaturalLanguage 框架，來進行中文的斷詞與詞彙嵌入。其他平台就算有 Swift 語言，不見得可以執行。

## 工具的用法

- 進入 tools/query 目錄
- 有兩種呼叫方式
  - `swift run QueryTool`: 列出所有語言的 ID
  - `swift run QueryTool <language_id> <query_text>` 傳入 ID 與查詢文字，會回傳一個 JSON 結構，裡面包含了斷詞後的每個 token，以及對應的詞彙與解釋。

範例句子：當用戶說「在南勢阿美語當中，怎麼說『我想吃飯』」，就可以查詢

```
swift run QueryTool 37 "我想吃飯"
```

AI 在不確定是什麼語言的狀況下，應該提示用戶應該指定語言，並透過 `swift run QueryTool` 告知用戶有哪些語言可以使用。指定之後，應該記住用戶指定的語言。

## 語法

台灣南島語系的語法通常以 VSO 為主。例如泰雅語、阿美語、排灣語等，通常會先交代「動作」，接著才是「是誰做的」以及「對象是誰」。AI 應該根據自己對台灣南島語系語言的知識，決定整個句子的語法。
